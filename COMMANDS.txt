Settings
1-4: Create model from designated obj
DELETE: Delete active model
LEFT/RIGHT: Rotate through models
X: Toggle X scaling
Y: Toggle Y scaling
Z: Toggle Z scaling
LEFT CTRL: Alternate rotation focus

Translation
NP_4: -X
NP_6: +X
NP_2: -Y
NP_8: +Y
NP_7: -Z
NP_9: +Z

Scaling
-: Scale active dimensions down
+: Scale active dimensions up

Rotation
I: Rotate X axis clockwise
K: Rotate X axis counterclockwise
J: Rotate Y axis clockwise
L: Rotate Y axis counterclockwise
U: Rotate Z axis clockwise
O: Rotate Z axis counterclockwise

Reflection
LEFT SHIFT: Reflects model in all active coordinates

Shear
V: Shear X -
N: Shear X +
B: Shear Y -
G: Shear Y +
F: Shear Z -
H: Shear Z +

Projection
P: Projects to active axes

F1: Animation
F2: Spline curve
